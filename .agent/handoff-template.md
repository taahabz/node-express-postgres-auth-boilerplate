# Agent handoff template

Use this format for final delivery notes.

## Summary
- What changed.
- Why it changed.
- Scope boundary (what intentionally did not change).

## Decision notes
- Key implementation decisions.
- Tradeoffs/alternatives considered (if relevant).

## Files touched
- [path/to/file](../path/to/file)

## Validation performed
- Commands/tests run.
- Result status.
- If tests were not run, explicit reason.

## Deployment impact
- Workflow/secret/env implications.
- Backward compatibility notes for existing template users.

## Follow-up actions
- Required secrets/config/manual steps.
- Suggested next tasks.

## Risk notes
- Known limitations or edge cases.
- Rollback hint if change affects deploy/runtime.
